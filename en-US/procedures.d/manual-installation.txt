
=== Manual Installation

This installation is based on the instructions at
http://www.eurekastreams.org/build-and-run/[].

==== Install prerequisite packages

These steps install numerous dependencies.

NOTE: These steps assume you are logged in as `root`.

. Install `maven2`
+
----
yum -y install maven2 maven2-manual
----

. Install `memcached`
+
----
yum -y install memcached
/sbin/service memcached start
/sbin/chkconfig memcached on
----

. Install Git
+
----
yum -y install git-all
----

. Install Postgres client and server
+
----
yum -y install postgresql postgresql-server
----

. Install Java from http://www.java.com/en[]
